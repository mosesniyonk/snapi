# Snapi

Snapi is a screenshot beautifier tool that helps you turn plain screenshots into polished, shareable visuals in seconds.

## ✨ Features

- Add beautiful backgrounds and gradients to screenshots
- Apply padding, rounded corners, and shadows
- Customize aspect ratio and canvas size
- Add optional captions or titles
- Fast, simple workflow built for creators and developers

## 🚀 Getting Started

### Prerequisites

- Node.js (LTS recommended)
- npm, yarn, or pnpm

### Installation

```bash
# clone the repository
git clone https://github.com/mosesniyonk/snapi.git
cd snapi

# install dependencies
npm install
```

### Run locally

```bash
npm run dev
```

Then open your browser at the local URL shown in the terminal.

## 🛠️ Build

```bash
npm run build
```

## 📦 Preview Production Build

```bash
npm run preview
```

## 💡 Usage

1. Upload or paste a screenshot.
2. Pick a style (background, spacing, border radius, shadow).
3. Adjust the layout until it looks perfect.
4. Export and share.
